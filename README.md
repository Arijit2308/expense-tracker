## Expense Tracker

A simple and efficient Expense Tracker web application to manage daily income and expenses. 
This project helps users track their spending habits and maintain financial awareness.

---

## Features

* Add income and expenses
* Filter transaction(all/income/expense)
* Delete transactions
* View balance in real-time
* Transaction history
* Simple and user-friendly interface
* Mobile responsive design
* Dark mode

## How to Run the Project

1. Clone the repository:

```bash

```

2. Open the project folder:

```bash
cd expense-tracker
```

3. Open `index.html` in your browser
